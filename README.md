# E2EE Messenger by Virat Rajput

End-to-End Encrypted Offline Messenger with 1-year persistence.

## Features
- 🔐 End-to-End Encryption
- 📱 Offline Support
- 🍪 1-Year Cookie Persistence
- 📁 File Upload with Encryption
- 🤖 Automation Support
- 👤 Facebook Integration
- 🎨 Beautiful UI with Virat Theme

## Quick Start

### Local Development
```bash
# 1. Clone repository
git clone <your-repo>
cd e2ee-messenger

# 2. Setup database
cd backend
python init_db.py

# 3. Install dependencies
pip install -r requirements.txt

# 4. Run server
python app.py

# 5. Open browser
http://localhost:5000
